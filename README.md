# ReverseCourse 反转课堂

反转课堂, 用项目驱动的方式引导学生进行学习.

孩子咳嗽老不好?多半是不想上补习班装的! ~~打一顿~~ 还不赶紧换个孩子喜欢的上课模式!

## 技术栈
Django & Vue.js

## 功能
- 小组
  - [ ] 学生自由结组

- 课题
  - [ ] 教师提交课题
  - [ ] 小组确认开始课题

- 文章(成果)
  - [ ] 小组提交 阶段成果
  - [ ] 小组提交 最终成果及展示图文
  - [ ] 教师评价打分

- 数据展示
  - [ ] 课题中各小组评分汇总
